Приглашение пользователя в организацию

В разделе "Управление организацией" во вкладке "Пользователи" администратор может пригласить пользователя из другой организации.

Нажмите на кнопку (1) выполните следующие шаги:

  1. Введите E-mail пользователя для добавления в организацию.
  2. Нажмите кнопку Пригласить (2).
  3. Пользователь получит доступ к организации.

Пользователь будет добавлен в организацию без прав.
