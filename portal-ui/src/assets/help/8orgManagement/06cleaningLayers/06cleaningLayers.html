Очистка дефолтных слоев

Вкладка Очистка предназначена для удаления временных геоданных данных с сервера, которые могут быть созданы на Geoserver при неудачном импорте файлов .shp. В этом разделе доступна функция Очистить дефолтные слои (1), которая удаляет ненужные файлы и освобождает место на сервере.
