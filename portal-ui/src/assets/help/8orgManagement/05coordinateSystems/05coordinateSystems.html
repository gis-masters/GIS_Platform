Системы координат

Вкладка "Системы координат" содержит доступные к использованию мировые и местные (пользовательские) системы координат.

Добавление пользовательской системы координат позволяет применять для геоданных специализированные, настраиваемые системы координат.

Для добавления пользовательской системы координат выполните следующие шаги:

  1. Нажмите на кнопку Создать пользовательскую систему координат (1).
  2. Введите текст системы координат в строку. Используйте файлы prj, выгруженные из QGIS или ArcGis.
  3. Нажмите кнопку Создать для добавления системы координат.

Пользовательская система координат будет добавлена в список и может быть использована в разделе "Настройки организации".
