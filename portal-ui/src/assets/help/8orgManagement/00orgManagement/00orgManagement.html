Управление организацией

* Страница «Управление организацией» доступна только для пользователя с правами администратора.

«Управление организацией» позволяет выдать или забрать права на проекты, библиотеки и наборы данных. А также открывает дополнительные возможности настройки организации.

Для перехода в «Управление организацией» нажмите:

Кнопку «Меню» и выберите «Управление организацией».

Доступны 5 вкладок:

  • Пользователи
  • Группы
  • Настройка организации
  • Системы координат
  • Очистка

При выборе одного из переключателя открываются дополнительные функции настойки организации
