Набор связанных файлов — это функция, которая показывает каких векторных файлов не хватает для размещения в проект.

При добавлении файла в карточку документа, отсутствующие файлы отображаются серым цветом, а добавленные в набор — синим цветом.

После добавления всех необходимых файлов вы можете скачать их в виде zip-архива 1 или разместить их в проекте 2.
