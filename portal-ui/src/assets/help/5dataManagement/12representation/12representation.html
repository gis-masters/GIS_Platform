Публикация векторной таблицы в проект

Публикация векторной таблицы на карте в виде слоя позволяет визуализировать информацию о географических объектах.

В слой можно добавлять новые объекты и редактировать существующие.

Для подключения в проект нажмите (1)

Доступен фильтр для поиска нужных проектов. Так же можно выбрать представление таблицы до её подключения.

Выбор представления данных влият на условные обозначения на карте и состав атрибутивных данных.

Для подключения в проект нажмите (2).

Векторная таблица подключена в указанный проект.
