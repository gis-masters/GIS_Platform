Редактирование схемы векторной таблицы

Редактирование схемы векторной таблицы позволяет настроить отображение информации для пользователей.

Для редактирования схемы выбранной векторной таблицы нажмите (1)

В окне редактирования структуры таблицы нажмите (2).

Измените свойства таблицы.

Нажмите "Предпросмотр" (3), чтобы увидеть обновленное отображение информации.

Нажмите (4) чтобы сохранить изменения.

Атрибуты векторной таблицы будут отображаться согласно изменной схеме.

Изменения будут применены только к выбранной векторной таблице.
