Табличный вид векторной таблицы

Табличный вид представляет собой удобный способ организации и представления атрибутивной информации о географических объектах, с помощью табличного вида можно легко фильтровать и искать нужные объекты на основе их атрибутов.

Переход в табличный вид (1).

В табличном виде можно открыть каждый объект отдельно (2) а также перейти к нему на карту если у него есть геоданные (3).

С помощью табличного вида можно легко фильтровать и искать нужные объекты на основе их атрибутов.

Для удобства можно настроить столбцы которые вам необходимы для просмотра у данной векторной таблицы.
