GML файл — векторная информационная модель документа территориального планирования.

GML файл может быть размещен на электронной карте в виде набора векторных слоев.

Для размещения загруженного GML на карте — нажмите на кнопку «Разместить в проекте» (1), выберите проект и нажмите на «Разместить в выбранном проекте» (2).

В результате документ будет опубликован на карте, его можно сопоставить с картографическими подосновами.

Переключатель (3) позволяет изменить направление координатных осей X и Y для векторных данных документа при размещении на карте.
