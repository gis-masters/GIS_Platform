Массовая загрузка КПТ из zip-архивов

Функция позволяет автоматизированно создавать и заполнять карточки документов на основе архивов для последующего импорта кадастровых данных в слои на карте.

Для массовой загрузки КПТ:

  1. В библиотеке «КПТ» создайте новую папку, если это необходимо.
  2. Перейдите в папку и нажмите (4).
  3. Заполните необходимые атрибуты, выберите zip-архивы и нажмите кнопку Загрузка (5).

Для каждого выбранного zip-архива будет создана и заполнена карточка документа с zip-файлом.
