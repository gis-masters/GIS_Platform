Свойства схем

Свойства схемы — это визуализированные правила форматирования для отображения данных

В свойствах схемы отображается тип свойства (1), название свойства (2), уникальный идентификатор (3) и дополнительные свойства поля (4).

Дополнительные свойства поля:

  • Поле обязательное для заполнения (5).
  • В поле нельзя внести изменения (6).
  • Скрытое поле не отображается для пользователя (7).
  • Дополнительное описание поля, отображается при наведении мышки на значок (8).
