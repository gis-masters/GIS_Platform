Наборы данных

Набор данных — это группа для упорядоченного хранения, комплексного назначения прав и массовой публикации векторных таблиц на карту.

Для перехода в раздел управления наборами данных, требуется на странице "Управление данными" выбрать «Наборы данных» (1) и нажать «Кнопку перехода в раздел» (2):

Для создания набора данных необходимо нажать на (1) ввести информацию по набору данных, сохранить.

В разделе «Наборы данных» представлен список всех наборов данных. Для удобства поиска и сортировки можно воспользоваться инструментами «Фильтр по названию» (3) и «Сортировка» (4).

Для того, чтобы открыть требуемый набор данных, нужно нажать (5). Перелистывание можно осуществлять с помощью стрелок вправо и влево (6).

В правом верхнем углу отображается название выбранного набора данных (7).

Для подключение всех векторных таблиц, которые находятся внутри набора данных нажмите (11).

Функция "Разрешения" (8) позволяет назначить права доступа к наборам данных.

Права предоставленные на уровне набора данных, наследуются всеми векторными таблицами внутри.

Для удаления набора данных нажмите «Удалить» (9), набор будет удален если в нем отсутствуют векторные таблицы.

Редактирование свойств набора данных (12) позволяет изменить его наименование или описание.
