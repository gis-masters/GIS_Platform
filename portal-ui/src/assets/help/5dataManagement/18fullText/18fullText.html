Полнотекстовый поиск

В управление данными доступен Фильтр по названию (1) и Полнотекстовый поиск (2).

Полнотекстовый поиск (2) — это метод поиска информации, при котором система умеет искать не только по ключевым словам, но и анализировать весь текст в таблицах и документах. Результаты поиска учитывают местонахождение пользователя в системе. Если запрос совпадает не полностью то система выдает наиболее подходящие результаты.

Для результатов поиска есть возможность открыть (3) найденные: документ, папку, векторный объект.

Для результатов поиска есть возможность перейти перейти к найденному объекту векторного слоя в проекте (4).
