«Заказ КПТ»

Функция «Заказ КПТ» предназначена для получения актуальных кадастровых сведений Росреестра в форме выписки "Кадастрового плана территории".

Для получения кадастровых данных КПТ Росреестра в библиотеке «КПТ» выберите заказ КПТ (4).

В диалоговом окне укажите номер кадастрового квартала. Если заказывается несколько кварталов, введите номера через запятую без пробелов.

  1. Нажмите кнопку Заказать (5).
  2. После размещения заказа в библиотеке «КПТ» будет создана новая папка, куда будут автоматически загружаться ZIP-архивы из ФГИС ЕГРН.
  3. За процессом заказа можно следить, используя механизм Задач.

Мониторинг выполнения задачи раздел «Задачи»

Процесс заказа КПТ может занять до двух недель. Следуйте шагам ниже, чтобы следить за выполнением системной задачи:

  1. Откройте раздел Задачи (6).
  2. Выберите созданную задачу по заказу КПТ (7).
  3. Отслеживайте статус выполнения задачи в колонке Статус (8). Когда статус изменится на Выполнено, процесс будет считаться завершенным.

После завершения задачи в библиотеке «КПТ» появится новая папка с ZIP-архивами, содержащими актуальные кадастровые данные из ФГИС ЕГРН.

Результатом выполнения задачи будет получение ZIP-архивов с кадастровыми сведениями, которые можно использовать для дальнейшей работы.
