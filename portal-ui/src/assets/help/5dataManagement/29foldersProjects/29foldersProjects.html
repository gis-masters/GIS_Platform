Создание папки проектов

Папки предназначены для группировки проектов, управления вложенностью и правами доступа.

Для создания новой папки выполните следующие шаги:

  1. Откройте раздел Управление данными.
  2. Во вкладке Проекты нажмите Создать папку проектов (1).
  3. Укажите название и описание (при необходимости).
  4. Нажмите Создать (2).

Созданная папка появится в начале списка.

Создание и перемещение проектов в папку доступны при наличии прав владельца или редактора соответствующей папки и проекта.
