Библиотека документов

Реестр документов

Реестр документов показывает библиотеку документов в табличном виде. Табличный вид нужен для систематизированного просмотра данных и их параметров.

Нажмите на кнопку (1), чтобы открыть реестр.

В режиме реестра доступны функции:
  • Экспортировать реестр (2) — выгружает информацию документов отобранных в реестре в файл формата .CVS, .XLSX.
  • Настройка столбцов (3) — вызывает диалоговое окно выбора видимых столбцов
  • Переход в иерархический вид (4) — возвращает иерархический вид библиотеки документов.

Нажмите (5) чтобы открыть меню управления документом.

Колонка «Полный пуь» (6) отображает путь до элемента. При наведении на "∨" открывается полный путь.
Так же путь отображается вверху страницы (7).

Нажмите на элемент пути чтобы перейти к нему.

Вверху страницы отображаются набор включенных фильтров и путь к объекту.

Выключите все активные фильтры нажав на крест (8) или отключайте фильтры по одному.

Нажмите на пункт в пути, чтобы перейти в другую папку.

Чтобы изменить ширину столбца в реестре — наведите мышь на серую границу в шапке реестра. Нажмите ЛКМ и потяните мышь в одну из сторон.

Использует реестр для быстрой и удобной работы с большим количеством данных.
