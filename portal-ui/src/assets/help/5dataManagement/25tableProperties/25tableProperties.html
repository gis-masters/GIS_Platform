Редактирование свойств векторной таблицы

Свойства векторной таблицы необходимы для быстрой идентификации ее основных параметров.

При выборе векторной таблицы в наборе данных в правой стороне экрана отображаются её свойства.

После создания векторной таблицы, вы можете изменить ее свойства, для этого нажмите (1).
