Перемещение проекта/папки в папку

Для перемещения проекта/папки в папку выполните следующие шаги:

  1. Выберите проект/папку из списка
  2. Нажмите Переместить (1).
  3. Выберите папку из списка для перемещения.
  4. Нажмите Выбрать (2).

Проект/папка будет перемещён в выбранное расположение.

Кнопка перемещения будет недоступной, если у пользователя права на проект/папку только на чтение.
