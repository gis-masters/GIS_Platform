Публикация DXF

DXF файл — открытый формат файлов для обмена графической информацией между приложениями.

Для визуализации и сопоставления данных DXF файл может быть размещен на электронной карте в виде набора векторных слоев.

Для размещения загруженного DXF на карте:

  1. Нажмите на кнопку «Разместить в проекте» (1);
  2. Выберите проект и «Систему координат» (2);
  3. Нажмите на «Разместить в выбранном проекте» (3).

В результате документ будет опубликован на карте, его можно сопоставить с картографическими подосновами и другими данными.
