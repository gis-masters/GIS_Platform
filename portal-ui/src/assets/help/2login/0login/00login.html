Для авторизации в геоинформационной системе Вам, требуется логин / адрес электронной почты (E-mail) и пароль выданной Вам учетной записи на портале.

Для входа в учётную запись на портале, требуется:

  1. Запустить приложение через браузер, для этого требуется ввести интернет-адрес геоинформационной системы в адресной строке браузера.
  2. Ввести данные от учетной записи.
  3. Нажать Войти для авторизации системы.
