Регистрация новой организации

Регистрация новой организации создаёт новое рабочее пространство для использования функционала картографической платформы.

Для регистрации организации нажмите на кнопку «Зарегистрироваться» на главной странице приложения.

Заполните данные о регистрируемой организации.

Выберите специализацию в зависимости от потребностей использования картографической платформы.

Выбор специализации определяет шаблоны данных для работы.

Нажмите кнопку «Зарегистрироваться».

Организация зарегестрирована.

Email, указанный при регистрации, является учётной записью владельца с полным доступом к управлению организацией.
