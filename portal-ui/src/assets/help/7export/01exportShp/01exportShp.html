Экспорт ESRI Shapefile

Экспорт слоя в формате ESRI Shapefile позволяет сохранить векторные данные проекта для последующего использования в других ГИС-системах. Результатом экспорта является ZIP-архив с SHP-файлом и сопутствующими файлами.

Экспорт слоя в Shapefile раздел Анализ и отображение данных

Для экспорта слоя в формате Shapefile выполните следующие шаги:

  1. Выберите нужный слой на карте.
  2. Откройте меню слоя (1).
  3. Нажмите кнопку Экспортировать в Shapefile (2).
  4. Выберите нужную систему координат (3).
  5. После завершения экспорта файл сохранится на ваш локальный компьютер.

В результате будет скачан ZIP-архив, содержащий экспортированный Shapefile с выбранным слоем.
