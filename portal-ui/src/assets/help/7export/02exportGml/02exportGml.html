Экспорт GML

Экспорт в формате GML используется для выгрузки пространственных данных в формате, совместимом со схемами территориального планирования.

Выгрузка GML

Для экспорта данных в формате GML выполните следующие шаги:

  1. Нажмите кнопку Выгрузка GML (1).
  2. Выберите необходимые слои (2) и схемы территориального планирования (3).
  3. Проверьте количество выбранных слоев в правом верхнем углу (4).
  4. Для отмены нажмите кнопку Отмена (5), для продолжения нажмите Экспорт (6).
  5. После завершения экспорта откройте окно Уведомления (7).
  6. Для удаления экспортированного GML нажмите (9).

В результате указанных действий будет сформирован архив с GML-файлом, содержащим выбранные пространственные данные. Для загрузки файла нажмите (8).
