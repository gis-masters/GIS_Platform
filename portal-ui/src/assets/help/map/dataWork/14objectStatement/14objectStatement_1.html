Выписка об объекте — это документ, содержащий характеристики выбранного объекта, включая его картографическое представление и координатное описание.

Для создания выписки об объекте, требуется:

  1. Открыть объект, выбрав его на карте либо из атрибутивной таблицы.
  2. Нажать ЛКМ (1)
  3. В параметрах печати ввести нужное название и выбрать фрагмент карты.
  4. Нажать «ОК» для сохранения выписки на локальный компьютер.

Создана выписка в формате PDF.
