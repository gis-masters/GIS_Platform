Карта. Работа с данными слоя (Атрибутивная таблица).

Множественное редактирование.

Множественное редактирование позволяет одновременно присвоить значения атрибутов выбранным объектам.

Для множественного редактирования объектов через атрибутивную таблицу:

  1. Откройте атрибутивную таблицу (1);
  2. Выделите объекты и нажмите на кнопку «Редактировать» (2);
  3. В появившейся панели нажмите «Изменить» (3) возле нужного поля и заполните атрибутивную информацию;
  4. Нажмите «Сохранить» (4).

В результате: выделенным объектам одновременно присвоены заданные атрибуты.
Значения в которые не внесли изменения остались такими же.
