Схема расположения объектов — это документ, объединяющий картографическое представление выделенных объектов с детализированной информацией о каждом объекте.

Для создания схемы расположения объектов требуется:

  1. Открыть атрибутивную таблицу слоя.
  2. Выделить объекты, которые должны быть включены в схему.
  3. Для удобство можно оставить только выделенные объекты. (1)
  4. Нажать ЛКМ (2).
  5. В диалоговом окне настроить параметры печати, указать название и выбрать фрагмент карты, который будет включен в схему.
  6. Нажать «ОК» для сохранения схемы на локальный компьютер.
>

Схема расположения объектов показывает объекты на карте, их легенду а также атрибутивную данные с выделенными объектами.

На локальный компьютер выгружен документ в формате PDF.
