Атрибутивная таблица

Атрибутивная таблица обеспечивает управление данными в табличном формате.

Таблица решает задачи:

  • Фильтрации объектов в атрибутивной таблице (1);
  • Выделения объектов в таблице и одновременно на карте (2);
  • Экспорта содержимого атрибутивной таблицы в файл Excel согласно фильтрам (3);
  • Редактирования/удаления данных (4);
  • Отображения/скрытия фильтрованных элементов на карте (5).

    • В поле Фильтр (1) существуют фильтры по строгому (=) и не строгому (≈) соответствию.
    (=) — идентичные значения.

    (≈) — содержится заданное значение.

    Нестрогий (≈) фильтр со знаками подстановки "%" и "."
    Знак "%" — любые символы.

    Знак "." — один любой символ.
