Режим изменения вершин

Функция позволяет изменять геометрию выбранных объектов на карте.

Включение режима изменения вершин:

  1. Выделите на карте несколько объектов, включая те, которые находятся в разных слоях.
  2. Откройте правую вкладку Выбранные объекты.
  3. Нажмите кнопку Режим изменения вершин (1).

Режим поддерживает одновременное изменение общих вершин нескольких объектов, а также использование прилипания для точного совпадения точек.

Действия в режиме изменения вершин:

  • Щелкните левой кнопкой мыши по объекту на карте.
  • Перемещайте вершины выбранного объекта, чтобы изменить его геометрию.
  • Дважды щелкните левой кнопкой мыши по вершине, чтобы удалить её.
  • Щелкните левой кнопкой мыши на линии выбранного объекта, чтобы добавить новую вершину.

Сохраните изменения геометрии объектов (2).

После сохранения геометрия объектов будет обновлена в соответствии с выполненными изменениями. Для выхода из режима нажмите (3).
