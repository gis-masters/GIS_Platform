Для создания (добавления) объекта на карте по координатам с помощью импорта из координат из CSV, требуется:

  1. Выполнить все предыдущие шаги по вводу данных (1-3) из предыдущего пункта "Создание объектов путём рисования на карте".
  2. Перейти на второй этап ввода данных, нажав на кнопку «Геометрия».
  3. Кликнуть на значок «Импорт координат контура из CSV».
  4. В появившемся окне, выбрать таблицу CSV и нажать кнопку «Открыть» и нажать кнопку «Сохранить».
