Для создания (добавления) объекта на карте по координатам из блокнота, требуется:

  1. Выполнить все предыдущие шаги по вводу данных (1-3) из предыдущего пункта "Создание объектов путём рисования на карте".
  2. Перейти на второй этап ввода данных, нажав на кнопку «Геометрия».
  3. Кликнуть на значок «Координаты контура как текст».
  4. В появившемся окне, ввести (вставить из буфера) данные координат и нажать на кнопку «Сохранить».
