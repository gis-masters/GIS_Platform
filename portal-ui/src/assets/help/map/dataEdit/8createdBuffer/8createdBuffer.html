Буферная зона — это область, созданная относительно границ объекта на указанное расстояние. Для векторных объектов размер буферной зоны может быть положительным (буфер создаётся снаружи) или отрицательным (буфер создаётся внутри).

Для создания буферной зоны:

  1. Выберите объект на карте.
  2. Нажмите кнопку Создать буфер (1) в правом меню.
  3. В открывшемся окне выберите Слой, в котором будет создана буферная зона.
  4. Укажите Размер буферной зоны в метрах.
  5. Нажмите Создать (2).
  6. Нажмите (3) чтобы сохранить объект.

Буферная зона отобразится на карте как новый объект.
