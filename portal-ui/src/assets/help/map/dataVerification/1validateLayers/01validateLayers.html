Проверка данных

Платформа даёт возможность проверить Ваши файлы на соответствие схемам Приказа №10 «Об утверждении Требований к описанию и отображению в документах территориального планирования объектов федерального значения, объектов регионального значения, объектов местного значения»

Для выполнения проверки данных нажмите на иконку «Проверка ошибок по Приказу» (1).

  • Прошедшие проверку слои, выделены зеленой «галочкой» (2).
  • Не проверенные – серым «восклицательным знаком» (3).
  • Содержащие ошибки – красным «восклицательным знаком» (4).
  • Количество объектов, содержащих ошибки – «красным цветом» (5).
