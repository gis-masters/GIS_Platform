Проверка данных

Просмотр ошибок

При нажатии на слой, содержащий ошибки, открывается таблица со списком объектов, содержащих ошибки. Таблица содержит поля:

  • ID объекта (1);
  • Название (2);
  • Количество ошибок в каждом объекте (3);
  • Кнопку «Показать объект на карте» (4);
  • Кнопку «Редактировать объект» (5);
  • Кнопку «Развернуть список» (6).

    • При нажатии (6) открывается таблица, со списком ошибок атрибутов слоя.

    Таблица содержит:

  • Название атрибута (7).
  • Обнаруженная ошибка заполнения атрибута (8).
    •

    Нажмите (5) чтобы сразу перейти к редактированию объекта.
