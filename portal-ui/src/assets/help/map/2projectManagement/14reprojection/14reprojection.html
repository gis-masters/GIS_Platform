Назначение системы координат для слоя

Функция перепроецирования слоя изменяет систему координат с пересчетом координат всех объектов слоя в новую проекцию для работы в другой системе координат.

Эта функция используется для отображения слоя в другой системе координат, отличной от исходной, что может потребоваться при работе с данными в разных проекциях.

Для выполнения перепроецирования выполните следующие шаги:

  1. Откройте меню Свойства слоя (1).
  2. Найдите поле Координатная система (2).
  3. Выберите систему координат из выпадающего списка или нажмите Выбрать другую (3).
  4. Слой автоматически перепроецируется в выбранную систему координат после сохранения изменений (4).

После настройки необходимо сохранить изменения слоя (5), перепроецированный слой будет отображаться в новой системе координат.
