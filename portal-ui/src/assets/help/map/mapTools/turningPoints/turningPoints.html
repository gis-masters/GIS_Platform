Поворотные точки

Функционал предназначен для формирования точек координат целевого объекта на карте.

Для включения и отображения поворотных точек выполните следующие шаги:

  1. Откройте меню аннотаций на верхней панели инструментов (1).
  2. При выборе одного объекта кнопка по созданию поворотных точек становится доступна
  3. Нажмите (2) для формирования поворотных точек.
  4. Для отключения поворотных точек нажмите (3).

Результат выполнения задачи: после выделения объекта на карте пользователь видит поворотные точки на всех вершинах объекта, которые могут быть использованы для анализа или вывода на печать.
