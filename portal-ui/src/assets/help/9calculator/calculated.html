"Калькулятор предоставления сведений" нужен для предоставления сведений, документов

Кнопка "Калькулятор предоставления сведений" вызывает калькулятор услуг (1)

(При первом вызове калькулятора необходимо заполнить реквизиты для оплаты услуг)

Калькулятор услуг содержит следующие кнопки:

  • Заполнить реквизиты(2) — форма содержит реквизиты для оплаты услуг (2)
  • Добавить услугу(3) — выбор списка услуг предоставления сведений из ГИСОГД
  • Очистить список услуг(4) — удаление всех услуг одним кликом
  • Печать квитанции(5) — формирует квитанцию в формате PDF

Для получения квитанции необходимо:

1. Заполнить реквизиты

2. Выбрать требуемые услуги в калькуляторе (3)

3. Выполнить печать квитанции (5)
